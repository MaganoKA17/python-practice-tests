## BASIC SECTION

### 1. sum_list
**Description:**  
You are given a list of numbers. Return the sum of all the numbers in the list.

### 2. count_even
**Description:**  
Count how many even numbers are present in a list.

### 3. capitalize_sentence
**Description:**  
Capitalize the first letter of each word in a sentence.

### 4. reverse_string
**Description:**  
Reverse the given string.

### 5. max_in_list
**Description:**  
Return the maximum number in a list. Return None if the list is empty.

## INTERMEDIATE SECTION

### 6. factorial
**Description:**  
Calculate the factorial of a non-negative integer.

### 7. remove_duplicates
**Description:**  
Remove duplicate elements from a list while preserving the original order.

### 8. sum_of_squares
**Description:**  
Return the sum of the squares of all numbers in a list.

### 9. sort_dict_by_values
**Description:**  
Sort a dictionary by its values and return a list of (key, value) tuples.

### 10. print_multiplication_table
**Description:**  
Return a 2D list representing the multiplication table up to n.

## ADVANCED SECTION

### 11. rotate_list
**Description:**  
Rotate a list to the right by k steps.

### 12. fibonacci_list
**Description:**  
Return a list of Fibonacci numbers up to the nth index.

### 13. flatten_list
**Description:**  
Recursively flatten a nested list of integers.

### 14. is_palindrome
**Description:**  
Recursively check if a string is a palindrome.

### 15. count_inversions
**Description:**  
Count the number of inversions in a list (pairs where a[i] > a[j] for i < j).
